# Balatro Card Art

An interactive Three.js recreation of Balatro's mesmerizing card and background effects. This project showcases the game's distinctive visual style using WebGL shaders and particle systems.

[Live Demo](https://ronald-luo.github.io/threejs-balatro/) | [Original Game on Steam](https://store.steampowered.com/app/2379780/Balatro/)

## Features

- Custom GLSL shaders recreating Balatro's fluid background
- Interactive 3D card with normal mapping and transparency
- Audio-reactive animations
- Particle effects emanating from card corners
- Responsive design with parallax effects

## Tech Stack

- Three.js for 3D graphics
- Custom GLSL shaders
- Web Audio API for music visualization
- Vite for development and building

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
